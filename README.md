# STEPS 


## INSTALL POETRY STEP

```
curl -sSL https://install.python-poetry.org | POETRY_HOME=~/.poetry python3 -
```


## ACTIVATE POETRY

Add below in the path

```
export PATH=$PATH:$HOME/.poetry/bin
```

Spin up poetry virtual env

```
$ poetry shell
```
